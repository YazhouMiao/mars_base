# MDC 组件使用帮助文档

## mdl-switch
```
<mdl-switch ng-model="dashModel.switch" 
            disable="true"
            display-label="{on: '开', 'off': '关'}">
</mdl-switch>
```
Options
```
    ng-model：必选项。switch绑定值，true or false对应switch的 on/off
    disable: 可选项。如果为true，switch会被禁用, 默认switch可用
    display-label: 可选性。如果没有提供，默认显示off/on
```
